# Advanced UI Component Static 代码背景介绍

## 项目概述

本模块提供 HarmonyOS 元服务高阶组件的静态版本，用于支持静态集成场景的元服务开发。

## 目录结构

```
advanced_ui_component_static/
└── fullscreenlaunchcomponent/
    └── @ohos.arkui.advanced.FullScreenLaunchComponent.ets  # 全屏嵌入式组件静态版
```

## 核心功能

- **FullScreenLaunchComponent**：FullScreenLaunch全屏嵌入式组件的静态版本（ArkTS 1.2标准），支持在静态集成场景下拉起元服务
- 与动态版本保持 API 兼容，提供一致的开发者体验
- 目前该部分不在我们仓库看护，ArkUI 团队目前还在负责维护中
